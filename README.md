# Mintlify Landing Page Clone

This project is a static clone of the Mintlify landing page, built using **pure HTML and CSS** as part of the **Chai Aur Code Web Development cohort assignment**.

The main objective of this project was to practice real-world UI cloning, layout structuring, and CSS styling without using JavaScript or any CSS frameworks.

---

## Live Demo

Live Link: https://mintlify-chai.netlify.app/ 
Original Website: https://mintlify.com

---

## Tech Stack

- HTML5  
- CSS3  
- No JavaScript  
- No frameworks or libraries  

---

## Features

- Sticky and translucent navigation bar  
- Hero section with background image and overlay  
- Clean typography inspired by Mintlify  
- Card-based layout sections  
- Trusted-by logos section  
- Layout built using Flexbox and Grid  

---

## What I Learned

- Structuring landing pages using semantic HTML  
- Using Flexbox and CSS Grid for layouts  
- Creating sticky headers and translucent effects  
- Handling background images and gradients  
- Improving spacing, alignment, and visual hierarchy  
- Cloning real-world websites using only HTML and CSS
